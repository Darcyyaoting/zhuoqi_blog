Shell发邮件

最近有需求要用shell发邮件，之前一直用

发附件邮件

但是昨天说要把文件发到正文当中

cat a.csv | sendmail -t

a.csv

#!/bin/bash

d1=”$(date +%Y%m%d -d ‘1 day ago’)”

echo ‘From: lizhuoqi lizhuoqi@baidu.com‘
echo ‘To: lizhuoqi lizhuoqi@baidu.com‘
echo ‘Content-type: text/html;charset=UTF-8’
echo ‘Subject: daishu_report_$d1’
echo ‘

‘
cat /home/…/daishu_$d1.csv|while read LINE
do
echo ‘‘
for LINE in $LINE
do
echo ‘‘
done
echo ‘‘
done
echo ‘
‘$LINE’
‘
`
经过一番探索，加强了对html和shell编程的掌握，总算基本做成了一个格式化的正文表格，后期可以继续优化html，以及加一些css
